=========================================================
              Landing Page de Concesionario Premium
=========================================================

Proyecto académico desarrollado para la materia de diseño
y optimización web. Consiste en la creación de una landing
page responsiva enfocada en autos Porsche 911 customizados 
de lujo. El objetivo central fue aplicar técnicas de 
optimización de imágenes, diseño responsivo y composición
visual con intención comercial.

=========================================================
1. OBJETIVO ACADÉMICO
=========================================================

• Desarrollar una landing page completa y responsiva.  
• Integrar un mínimo de diez imágenes optimizadas para web.  
• Demostrar uso correcto de HTML, CSS y JavaScript.  
• Aplicar técnicas de optimización de imágenes (WebP, compresión).  
• Mantener coherencia entre diseño, contenido y marca.  
• Realizar composición visual con intención publicitaria.  
• Utilizar CSS moderno: Grid, Flexbox y media queries.  

=========================================================
2. PROPÓSITO DEL SITIO
=========================================================

El sitio representa un showroom digital ficticio llamado 
"911 Heritage". Su finalidad es comunicar exclusividad, 
diseño contemporáneo y personalización avanzada de modelos 
Porsche 911, al tiempo que invita al usuario a solicitar 
información o agendar una cita.

La página se centra en:

• Exhibición de modelos exclusivos.  
• Personalización interior y exterior.  
• Historia y herencia del Porsche 911.  
• Ediciones especiales de alto desempeño.  
• Transmisión de estética premium.  

=========================================================
3. TECNOLOGÍAS UTILIZADAS
=========================================================

• HTML5  
• CSS3 (Grid, Flexbox, diseño responsivo)  
• JavaScript (interacciones y menú móvil)  
• Imágenes WebP optimizadas  
• Diseño mobile-first  

=========================================================
4. OPTIMIZACIÓN DE IMÁGENES
=========================================================

Las imágenes fueron:

• Convertidas a formato WebP para rendimiento óptimo. 
• Mantuvimos las imagenes JEPG necesarias para obtener mejores colores y calidad.
• Redimensionadas para menor carga sin perder nitidez.  
• Pruebas de visualización en desktop y mobile.  
• Seleccionadas por coherencia con la marca Porsche.  
• Utilizadas en múltiples secciones cumpliendo el mínimo requerido.  

=========================================================
5. SECCIONES DEL SITIO
=========================================================

• Hero / Landing principal  
• Diseño Atemporal  
• Exhibición & Interiores (dos columnas)  
• Modelos de Lujo 2025 (inversión visual)  
• Porsche hecho a tu medida
